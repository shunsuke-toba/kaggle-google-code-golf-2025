# Task 392: 0 矩形から拡張する交互枠塗り

## 問題文

10×10 のグリッド `g` が与えられます。各セルは 0〜9 の色を表します。黒色(0)のセルの中に、ちょうど 1 つだけ連結かつ完全な矩形を成す領域があります。

この矩形の幅を `w`、高さを `h` とし、`n = max(w, h)` とします。矩形の周囲 1 マスを取り囲むように、同じ色 `c` からなる正方形の枠が存在します (枠の内側は塗られていません)。

この枠を中心を変えずに外側へと拡張し、幅 1 の枠を `n` マス間隔で同心状に追加します。色は内側から順に

```
灰色(5), 枠色 `c`, 灰色(5), 枠色 `c`, ...
```

と交互になります。元の黒い矩形も灰色に塗り替えてください。

## 入力形式

- `g` は 10×10 の二次元リスト
- 要素は 0〜9 の整数

## 出力形式

- 同じサイズの二次元リスト
- 上記規則に従い塗り替えた結果

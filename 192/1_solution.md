# Task 192: 孤立セルの多数決塗り

## 問題

0〜9 の数値で塗られたグリッドが与えられます。0 は黒 (背景) を表します。

次の手順でグリッドを変換してください。

## ルール

1. 0 以外のセルのうち，上下左右に同じ値を持つセルが一つもないものを「孤立セル」とする。
2. 孤立セルについて，上下左右 4 マスの色を調べる。範囲外は 0 とみなす。
3. もっとも多く現れる色でそのセルを塗り替える。
4. もし 2 色が 2 回ずつ現れ，かつ上下の色が等しく左右の色も等しい（上下と左右は異なる）場合は 0 で塗り替える。
5. 上記以外で最頻色が複数ある場合は値が大きい色を選ぶ。0 は最も小さい。

孤立セル以外は変更しません。

## 出力

変換後のグリッドを返します。

## 例

入力

```
0 3 0
0 8 3
0 0 0
```

出力

```
0 3 0
0 3 3
0 0 0
```

入力

```
0 0 0
9 8 9
0 0 0
```

出力

```
0 0 0
0 0 0
0 0 0
```

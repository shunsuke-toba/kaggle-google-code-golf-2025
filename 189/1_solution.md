# Task 189: パレットで図形を着色

## 問題文

9×9 のグリッド `g` が与えられる。このグリッドには次の特徴がある。

- 1 行と 1 列がすべて `8` で塗られており，その交点でグリッドが 4 つの領域に分かれる。
- ある領域は 2×2 の大きさで，`0`・`3`・`8` 以外の数値が 4 つ置かれている。これを**パレット**と呼ぶ。
- パレットと対角に位置する 6×6 の領域には値 `0` と `3` のみが現れ，6×6 を 3×3 の 4 ブロックに分割すると各ブロック内の `3` が何らかの形を描いている。

求める出力はこの 6×6 の領域を次の規則で着色したものである。

1. 3×3 の各ブロックで `3` をパレットの対応する値に置き換える。
   - パレットの 4 つの値は位置に応じて左上・右上・左下・右下のブロックに対応する。
2. `0` はそのまま `0` とする。

以上の操作で得られる 6×6 のグリッドを出力せよ。

## 入力形式

- 9×9 の整数グリッド `g`
- 各要素は 0 以上 9 以下

## 出力形式

- 6×6 の整数グリッド
- 値はパレットの色または `0`

## 例

入力例

```
2 4 8 0 0 0 0 0 0
1 6 8 0 0 0 0 0 0
8 8 8 8 8 8 8 8 8
0 0 8 0 3 0 0 3 0
0 0 8 3 3 3 3 3 3
0 0 8 0 3 0 0 3 0
0 0 8 0 3 0 0 3 0
0 0 8 3 3 3 3 3 3
0 0 8 0 3 0 0 3 0
```

出力例

```
0 2 0 0 4 0
2 2 2 4 4 4
0 2 0 0 4 0
0 1 0 0 6 0
1 1 1 6 6 6
0 1 0 0 6 0
```

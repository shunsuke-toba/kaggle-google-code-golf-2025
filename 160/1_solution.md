# Task 160: 青い 3x3 を赤いクロスに変換

## 問題

グリッドには 0(黒),1(青),2(赤)の色が使われる。いくつかの青の連結成分が存在し、その中で**3x3 の正方形を成し中心が黒のもの**を見つける。
該当する 3x3 領域は中心を起点とした十字型に変形し、色を赤(2)に変更する。元の 3x3 の 4 隅は黒(0)に置き換える。
他のセルはそのまま。

## 例

入力:

```
1 1 1
1 0 1
1 1 1
```

出力:

```
0 2 0
2 2 2
0 2 0
```

## 解法のヒント

各セルを中心とした 3x3 領域を調べ、周囲 8 マスが青で中心が黒ならその領域を上記のように書き換える。

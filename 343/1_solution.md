# Task 343: 横方向へのパターン補完

## 問題

H\*W のグリッド `g` が与えられる。左側にはいくつかの色付きセルが並んでおり、これは高さ H, 幅 n のパターンを横方向へ繰り返して描いたものの一部である。右側の未完成な部分は 0 で埋められている。なお x=0 がパターンの先頭とは限らず途中から始まっている場合もある。

このパターンの幅 `n` を推定し、同じパターンが途切れずに W 列目まで続くように 0 を置き換えよ。

## 入力

- `g`: H 行 W 列の整数グリッド (0-9)。右側には未完成部分として 0 が含まれる。

## 出力

- 推定したパターンを横方向に繰り返し、W 列全体を埋めたグリッドを返す。

## 例

入力:

```
0 0 0 0 0
1 2 1 2 0
3 4 3 4 0
```

出力:

```
0 0 0 0 0
1 2 1 2 1
3 4 3 4 3
```

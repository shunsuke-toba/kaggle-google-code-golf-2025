# Task 270: 起点による色の引き寄せ

## 問題文

0 を背景とする H×W グリッドが与えられる。グリッドには値 1,2,3,7 が含まれる。

- 1 は「青の起点」であり，同じ行または列にある 7 を引き寄せる。
- 2 は「赤の起点」であり，同じ行または列にある 3 を引き寄せる。

引き寄せの規則は以下の通りである。

1. 各起点について，上下左右の 4 方向を調べる。
2. その方向に対応する色(1→7, 2→3)が 1 つでも存在すれば，
   起点の隣接マス(同じ方向)にその色を配置する。
   途中に他の色があっても無視してよい。
3. それ以外のマスは 0 になる。起点自体の値は変化しない。

## 入力形式

- 0,1,2,3,7 からなる H×W グリッド。
- 1 と 2 はそれぞれ 1 個ずつ存在する。

## 出力形式

- 上記の引き寄せ操作を適用した後の H×W グリッド。

## 例

入力

```
000300000000000
300200003000000
000000000007000
000300000000000
000007000001007
000000000007000
```

出力

```
000300000000000
003230000000000
000000000000000
000000000007000
000000000071700
000000000007000
```

# Task 394: タイルパターンの復元

## 問題文

数字 (0-9) からなる H×W のグリッド `g` が与えられます。
このグリッドには 0 で塗られた長方形の領域が 1 つ存在し、そこだけ本来の色が隠されています。

グリッド全体は、ある最小のタイルを上下左右に繰り返し敷き詰めたパターンから作られています。
既知の部分からこのタイルを推測し、隠された領域を復元してください。

- 入力: 2 次元配列 `g`
- 出力: マスキングされた領域と同じ大きさの 2 次元配列
  （復元した値のみを返す）

0 はマスキングを表し、他の位置には現れません。

## 例

### 入力

```
2 1 2 1 2
1 1 1 1 1
2 1 2 1 2
0 0 1 1 1
0 0 2 1 2
```

### 出力

```
1 1
2 1
```

上の例では 2×2 の領域が隠されており、
タイルパターンを推測すると `[ [2,1],[1,1] ]` が繰り返されていることがわかります。
このタイルを用いて隠された部分を復元すると、出力のようになります。

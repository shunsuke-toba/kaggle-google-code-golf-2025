# Task 398: 斜め複製拡張

## 問題

1x5 のグリッドが与えられる。各セルには 0 (空白) または色番号が入る。0 でないセルの個数を n とする。

幅と高さが 5n の正方形グリッドを作り、次のように色を配置せよ。

- 最下行の左端に元の 1x5 行をそのまま配置し、余った部分は 0 で埋める。
- 1 行上に移るごとに、前の行を右へ 1 マスずつずらして配置する。はみ出した部分は捨てる。

結果として、元の色は左下から右上へ向かう対角線上に並ぶ。

## 入力

- 1x5 のグリッド `g`。

## 出力

- 上記ルールで拡張した (5n)x(5n) グリッド。

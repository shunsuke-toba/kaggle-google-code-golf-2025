# Task 168: L 字ブロックからの対角線延長

## 問題

グリッドには 0 と複数の色が配置されている。ある色について、2×2 領域のうち 3 マスだけがその色で 1 マスが背景色 0 である部分を考える。これは欠けた L 字ブロックである。

各欠けた L 字ブロックに対し、欠けている角からその角が空いている方向へ同じ色の対角線をグリッドの端まで描く。ただし欠けているマス自体は塗らない。

## ルール

1. 2×2 領域で同じ色が 3 マス、残り 1 マスが 0 のものを全て見つける。
2. 欠けているマスを中心に、空いている方向へ 1 マス進んだ位置から同じ色で対角線を引く。
3. 対角線はグリッドの端に達するまで伸ばす。
4. 既存の色付きマスはそのまま残し、欠けたマスは塗らない。

## 例

### 入力

```
0000000000
0000000000
0700000000
0770000000
0000007700
0000000700
0000000000
0000000000
0000000000
0000000000
```

### 出力

```
0000700000
0007000000
0700000000
0770000000
0000007700
0000000700
0000070000
0000700000
0007000000
0070000000
```
### 実装方法
- 以下を4回繰り返す
- k=-1とする
- すべてのマスを上から探索する
- i+j==yならそのマスをxで塗る
- 上、右、右上のマスが全て0でない場合その数字をxとし、y=i+jとする
- 盤面全体を回転する

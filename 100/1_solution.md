# Task 100: 面積が大きい長方形の色

## 問題文

H×W のグリッド `g` には 0 以外の色で描かれた長方形の枠がちょうど 2 つ存在します。
枠は互いに交差せず、内部は塗りつぶされていません。2 つの長方形の面積は異なるので、より
大きな面積を持つ長方形の色 `c` を求め、その色のみで構成された 2×2 の正方形を出力して
ください。

## 入力形式

- 0〜9 の整数からなる二次元配列 `g`

## 出力形式

- 2×2 の二次元配列。すべての要素が色 `c`

## 例

入力と出力の例は `google-code-golf-2025/task100.json` の `train` セクションを参照してください。

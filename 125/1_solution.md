# 問題の考察ポイントと解き方

## 考察ポイント

1. **ピンク色(6)の連結成分が矩形の枠を形成している**

   - グリッド内に複数のピンク色の連結成分が存在
   - 各連結成分は閉じた矩形の枠を形成

2. **2 段階の変換処理**

   - 第 1 段階: ピンクの枠で囲まれた内部領域の処理
   - 第 2 段階: ピンクセルの周辺 8 マスの処理

3. **色の変換ルール**
   - 内部の水色(8) → 黄色(4)
   - ピンク周辺の水色(8) → 緑(3)
   - ピンク(6)自体は変更なし

## 解くための手続き

1. **結果グリッドの初期化**

   - 入力グリッドをコピーして結果グリッドを作成

2. **ピンクの連結成分を探索**

   - 各ピンク(6)セルから開始して 4 連結で BFS/DFS
   - 訪問済みフラグで重複探索を防ぐ
   - 各連結成分の境界(min_y, max_y, min_x, max_x)を記録

3. **内部領域を黄色で塗りつぶし**

   - 各連結成分について、境界内の非ピンクセルを検査
   - 水色(8)のセルを黄色(4)に変換

4. **ピンク周辺を緑化**

   - 全てのピンク(6)セルについて 8 近傍を確認
   - 水色(8)のセルを緑(3)に変換

5. **変換後のグリッドを返却**

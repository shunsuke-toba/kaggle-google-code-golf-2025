# 問題の考察ポイントと解き方

## 考察ポイント

1. **ピンク色(6)の連結成分が矩形の枠を形成している**

   - グリッド内に複数のピンク色の連結成分が存在
   - 各連結成分は閉じた矩形の枠を形成

2. **2 段階の変換処理**

   - 第 1 段階: ピンクの枠で囲まれた内部領域の処理
   - 第 2 段階: ピンクセルの周辺 8 マスの処理

3. **色の変換ルール**
   - 内部の水色(8) → 黄色(4)
   - ピンク周辺の水色(8) → 緑(3)
   - ピンク(6)自体は変更なし

## 解くための手続き
- 8のマスを全部4にする
- g[0][0]を8にする
- 全部のマスをみて
    - 4,6のマスでないかつ周囲8マスに6のマスがあるなら3にする
    - 3,6のマスでないかつ周囲8マスに8のマスがあるなら8にする

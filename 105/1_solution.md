# Task 105: 青ブロックの矩形枠と最多線の強調

## 問題の考察ポイントと解き方

### 考察ポイント

1. **青ブロックの外接矩形**

   - グリッド内の青（色 1）のブロックすべてを囲む最小の矩形を特定
   - min_x, max_x, min_y, max_y を求める

2. **矩形の枠線の描画**

   - 外接矩形の 4 辺（上辺、下辺、左辺、右辺）を塗る
   - 元々青の位置は青のまま、黒の位置は赤（色 2）に変える

3. **最多ブロックを含む線の検出**
   - 矩形内部の各横線・縦線について、**枠を除いた内部のみ**の青ブロック数をカウント
   - 全体（横線と縦線を合わせて）で最も多くの青ブロックを含む線を特定
   - 最多の線をすべて塗る（複数ある場合はすべて）

### 解くための手続き

1. **青ブロックの座標を収集**

   - グリッド全体を走査し、青（色 1）のセルの座標をすべて記録

2. **外接矩形の計算**

   - 青ブロックの座標から min_y, max_y, min_x, max_x を求める

3. **矩形の枠を塗る**

   - 上辺 (y=min_y, x=min_x ～ max_x)
   - 下辺 (y=max_y, x=min_x ～ max_x)
   - 左辺 (x=min_x, y=min_y ～ max_y)
   - 右辺 (x=max_x, y=min_y ～ max_y)
   - 各位置で：元が青なら青のまま、元が黒なら赤に変更

4. **内部の線分の青ブロック数をカウント（枠を除く）**

   - 横線：各 y (min_y+1 ～ max_y-1) について、**x (min_x+1 ～ max_x-1) の範囲**で青ブロック数を数える
   - 縦線：各 x (min_x+1 ～ max_x-1) について、**y (min_y+1 ～ max_y-1) の範囲**で青ブロック数を数える

5. **全体で最多の線を特定し塗る**
   - 横線と縦線を合わせて、最も多くの青ブロックを含む線を見つける
   - 最多が 0 より大きい場合：
     - 最多の青ブロック数を持つすべての線を塗る（枠を含む全幅/全高）
     - 元が青なら青のまま、元が黒なら赤に変更
   - 最多が 0 の場合：何も塗らない

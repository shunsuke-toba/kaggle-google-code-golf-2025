# task123: シェビチェフ距離による 10x10 グリッド拡張

## 問題の考察ポイント

1. **入力の構造**

   - 5x5 のグリッドが与えられる
   - 最初の行に複数の色（非黒色）が含まれる
   - 黒（0）は背景色として扱われる

2. **出力パターン**

   - 10x10 のグリッドに拡張される
   - シェビチェフ距離（max(y, x)）に基づいて色が配置される
   - 原点(0,0)からの距離ごとに異なる色で塗られる

3. **色の割り当て規則**
   - 入力の最初の行から非黒色を順番に抽出
   - 距離 0 から順番に、抽出した色を循環的に割り当て
   - 距離 d = max(y, x)の位置には、colors[d % len(colors)]の色を配置

## 解くための手続き

1. **色リストの作成**

   - 入力グリッドの最初の行（g[0]）から非黒色（0 以外）を抽出
   - 順序を保持したまま色リストを作成

2. **10x10 グリッドの初期化**

   - 全て 0 で初期化した 10x10 のグリッドを作成

3. **シェビチェフ距離による塗りつぶし**

   - 各座標(y, x)について、距離 d = max(y, x)を計算
   - その位置に色リストの d 番目の色を配置（モジュロで循環）
   - r[y][x] = colors[d % len(colors)]

4. **結果の返却**
   - 完成した 10x10 グリッドを返す

# Task 134: 正方形領域の穴埋め

## 問題文

0 〜 9 の整数からなる H×W グリッド `g` が与えられます。0 は黒を表します。

グリッドには 0 以外の色がちょうど 2 種類含まれており、それぞれの色のマスをすべて含む最小の長方形を考えます。面積がより小さい方の長方形は正方形で、その一辺の長さは 3 の倍数です。

この正方形領域を 3×3 の等しい小領域に分割します。各小領域がすべて正方形の色で塗られていれば、出力グリッドの対応する位置をもう一方の色で塗ります。そうでなければ 0 を置きます。

## 入力

- 2 次元配列 `g`

## 出力

- 3×3 の 2 次元配列

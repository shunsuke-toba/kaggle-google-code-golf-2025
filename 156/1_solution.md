# Task 156: 塗り分けられた長方形の内部

## 問題文

グリッドには背景色 0 の上に、色 4 で塗られた長方形が 2 つ存在します。
これらは互いに重ならず、各長方形は縦横が一直線に揃った塗りつぶし領域です。

各長方形について、境界(周囲 1 セル幅)はそのまま色 4 を保ち、
内部のセルを以下の規則で塗り替えてください。

- 面積が小さい長方形の内部を色 1 で塗りつぶす。
- 面積が大きい長方形の内部を色 2 で塗りつぶす。

同じ大きさの長方形は出題されません。

## 入力形式

- 0–9 の整数からなる二次元配列 `g`

## 出力形式

- 変換後の二次元配列

## 例

例えば次の入力 `g` の場合：

```
0 4 4 4 4 0
0 4 4 4 4 0
0 4 4 4 4 0
0 4 4 4 4 0
0 0 0 0 0 0
0 0 4 4 4 4
0 0 4 4 4 4
0 0 4 4 4 4
0 0 4 4 4 4
```

出力は以下のようになります：

```
0 4 4 4 4 0
0 4 1 1 4 0
0 4 1 1 4 0
0 4 4 4 4 0
0 0 0 0 0 0
0 0 4 4 4 4
0 0 4 2 2 4
0 0 4 2 2 4
0 0 4 4 4 4
```

小さい正方形の内部が 1、大きい長方形の内部が 2 に置き換わっていることがわかります。

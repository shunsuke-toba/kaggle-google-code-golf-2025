# Task 175: 対角線対称グリッドの復元

## 問題

0 から 9 の整数で構成された正方形グリッド g が与えられます。0 は色が失われた「黒」を表します。
このグリッドは本来、左上 (0,0) から右下 (n-1,n-1) への対角線を軸に対称であり、
0 のセルを元の色に復元してください。

## 復元規則

- 0 でないセルはそのまま。
- セル (r,c) が 0 の場合:
  - r==c なら g[0][0] の色で塗る。
  - それ以外なら対称位置 (c,r) の色で塗る。
    - その位置も 0 なら g[0][1] (≡ g[1][0]) の色で塗る。

入力は少なくとも 2×2 の正方形で、g[0][0] と g[0][1] は 0 ではないとします。

## 入出力

- **入力**: n×n の 2 次元配列 g。
- **出力**: 上記規則を適用した n×n の 2 次元配列。

# Task 093: グレー矩形の方向別延長

## 問題の考察ポイントと解き方

### 考察ポイント

1. **グレー矩形の境界判定**

   - グリッド内のグレー(5)のセルから矩形の範囲（最小/最大の行・列）を特定
   - 矩形の 4 辺（上・下・左・右）の位置を把握

2. **方向別の探索ルール**

   - 各辺から外側方向にのみ探索（上辺 → 上、下辺 → 下、左辺 → 左、右辺 → 右）
   - 探索は盤面の端まで継続

3. **カウント条件**

   - 黒(0)とグレー(5)以外の値を持つセルのみをカウント
   - つまり、値が 1,2,3,4,6,7,8,9 のセルが対象

4. **延長ルール**
   - カウントした数だけ、その方向にグレー(5)を延長
   - グリッドの残り全ては黒(0)で塗りつぶす

### 解くための手続き

1. **グレー矩形の範囲を取得**

   - 全セルを走査してグレー(5)の位置を記録
   - min_row, max_row, min_col, max_col を算出

2. **上辺からの延長処理**

   - 列ごとに(min_col ～ max_col)処理
   - 0 行目から min_row-1 行目までを探索
   - 非 0 非 5 のセル数をカウント
   - min_row-1 から上にカウント数分グレーを配置

3. **下辺からの延長処理**

   - 列ごとに(min_col ～ max_col)処理
   - max_row+1 行目から最終行までを探索
   - 非 0 非 5 のセル数をカウント
   - max_row+1 から下にカウント数分グレーを配置

4. **左辺からの延長処理**

   - 行ごとに(min_row ～ max_row)処理
   - 0 列目から min_col-1 列目までを探索
   - 非 0 非 5 のセル数をカウント
   - min_col-1 から左にカウント数分グレーを配置

5. **右辺からの延長処理**

   - 行ごとに(min_row ～ max_row)処理
   - max_col+1 列目から最終列までを探索
   - 非 0 非 5 のセル数をカウント
   - max_col+1 から右にカウント数分グレーを配置

6. **最終グリッドの構築**
   - 初期状態：全セルを黒(0)で初期化
   - 元のグレー矩形領域を復元
   - 各方向への延長部分を追加

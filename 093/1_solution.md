# Task 093: グレー矩形の方向別延長

## 問題の考察ポイントと解き方

### 考察ポイント

1. **グレー矩形の境界判定**

   - グリッド内のグレー(5)のセルから矩形の範囲（最小/最大の行・列）を特定
   - 矩形の 4 辺（上・下・左・右）の位置を把握

2. **方向別の探索ルール**

   - 各辺から外側方向にのみ探索（上辺 → 上、下辺 → 下、左辺 → 左、右辺 → 右）
   - 探索は盤面の端まで継続

3. **カウント条件**

   - 黒(0)とグレー(5)以外の値を持つセルのみをカウント
   - つまり、値が 1,2,3,4,6,7,8,9 のセルが対象

4. **延長ルール**
   - カウントした数だけ、その方向にグレー(5)を延長
   - グリッドの残り全ては黒(0)で塗りつぶす

### 解くための手続き

- 以下の一連の操作を4回行う
- 14行見て、各行の0~6列までの0でないマスの個数をxとする
- 0~6列までを全部0にしたあと6列目から左x列分を5にする
- 90度回転

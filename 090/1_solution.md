# Task 090: 最大矩形の探索と塗りつぶし

## 問題の考察ポイントと解き方

### 考察ポイント

1. **空きマスの判定**

   - 黒(0)のセルのみが空きマスとして扱われる
   - 他の値（1 以上）は全て障害物として扱う

2. **矩形の制約条件**

   - 矩形のサイズは高さ >= 2 かつ 幅 >= 2 を満たす必要がある
   - 矩形内の全てのセルが黒(0)である必要がある

3. **最大矩形の定義**

   - 面積（高さ × 幅）が最大となる矩形を探す
   - 同じ面積の矩形が複数ある場合、どれを選んでも良い

4. **出力形式**
   - 元のグリッドの最大矩形部分のみを紫(6)で置き換える
   - それ以外の部分は元の値を保持

### 解くための手続き

1. **全ての矩形候補を列挙**

   - 各セル(i,j)を左上隅とする
   - 右下隅(r,c)を全探索（i<r, j<c）
   - 高さ h = r-i+1、幅 w = c-j+1 を計算

2. **矩形の有効性判定**

   - h >= 2 かつ w >= 2 の確認
   - 矩形範囲内の全セルが黒(0)であることの確認
   - 一つでも黒以外があれば無効

3. **最大矩形の記録**

   - 有効な矩形の面積を計算（h × w）
   - 現在の最大面積より大きければ更新
   - 矩形の座標（左上と右下）を保存

4. **結果グリッドの生成**
   - 元のグリッドを完全にコピー
   - 最大矩形の座標範囲内を紫(6)で塗りつぶし
   - 最大矩形が存在しない場合は元のグリッドをそのまま返す

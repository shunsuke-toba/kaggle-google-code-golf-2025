# Task 265: 2x2 領域の赤塗り

## 問題

0 から 9 の色で塗られた H×W グリッド `g` が与えられます。色 `5` は「グレー」、色 `2` は「赤」を表します。

任意の 2×2 の領域を調べ、**その 4 マスの中にグレー(5)が 1 つも存在しない** 場合には、その 4 マス全てを赤(2)で塗りつぶしてください。複数の領域が重なる場合は同時に処理します。

# 注意
0, 0, 5
0, 0, 0
0, 0, 0
0, 0, 5

は

2, 2, 5
2, 2, 0
2, 2, 0
2, 2, 5

のように、最大長方形だけを塗る？

## 出力

変換後のグリッドを返してください。

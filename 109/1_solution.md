# Task 109: 十字分割パターン変換

## 問題の考察ポイントと解き方

## 考察ポイント

1. **十字型分割線の検出**: 単一色で構成される行と列を見つける
2. **パターン抽出**: 左上の象限からパターンを取得
3. **色の置換**: パターン内の色を分割線の色に変換
4. **4 種類の変換**:
   - 左上: 元のパターン（色置換済み）
   - 右上: Y 軸反転（左右反転）
   - 左下: X 軸反転（上下反転）
   - 右下: X 軸・Y 軸両方反転

## 解くための手続き

1. 入力グリッドから単一色の行と列を探し、分割線の位置と色を特定
2. 左上の象限（分割線より前の部分）を抽出
3. パターン内の非ゼロ色を分割線の色に置換
4. 出力グリッドのサイズを計算（パターンサイズ ×2）
5. 各象限に変換したパターンを配置:
   - (0,0): そのまま配置
   - (0,w): 左右反転して配置
   - (h,0): 上下反転して配置
   - (h,w): 上下左右反転して配置

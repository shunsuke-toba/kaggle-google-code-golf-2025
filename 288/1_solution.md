# Task 288: 対角線塗りつぶし

## 問題

H\times W のグリッドが与えられます。各セルは整数で色を表し、0 は空き(黒)を示します。
以下の手順でグリッドを書き換えてください。

1. **基準色**を最下行の中央セル (列番号 \(W//2\)) の色とする。
2. 下から 2 行目の行を左から右へ走査する。
   - セルが 0 以外で、その左隣が 0 の場合、左上方向へ進みながら基準色で塗りつぶす。
   - セルが 0 以外で、その右隣が 0 の場合、右上方向へ進みながら基準色で塗りつぶす。
   - いずれも盤面外に出るまで進み、通過したセルを基準色で上書きする。開始セル自身は変更しない。
3. 上の操作後のグリッドを出力する。

## 入出力

- 入力・出力はいずれも同じ大きさの 2 次元配列で表す。
- 各要素は整数。

## 例

入力

```
0 0 0
0 2 0
2 4 2
```

出力

```
4 0 4
0 2 0
2 4 2
```

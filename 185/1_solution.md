# Task 185: 色付き交点の抽出

## 問題

0–9 の数値で構成されたグリッドが与えられる。
グリッドにはある 1 色の直線が縦横に 3 等間隔で引かれており、他のセルは 0 である。
それらの直線の交点のいくつかが別の色で塗られている。
この交点群を使って 3×3 の図形を復元せよ。

## ルール

1. グリッド全体で最も多く現れる非 0 色を線の色とする。
2. 線の色でも 0 でもないセルを集め、その最小/最大の x,y から軸に平行な外接矩形を得る。
   この矩形は各辺が 3 の倍数の長さを持つ。
3. 辺の長さを 3 で割った値を d とする。
   矩形左上を原点とし、x と y をそれぞれ d 刻みで 3 つずつ取る。
4. 各 (x,y) について、四隅
   (x,y),(x+d,y),(x,y+d),(x+d,y+d) の色を調べる。
   四隅が同じ色かつ線の色でないならその色を出力グリッドの対応するセルに置き、
   そうでなければ 0 を置く。

## 出力

上記の手順で得られる 3×3 のグリッドを返す。

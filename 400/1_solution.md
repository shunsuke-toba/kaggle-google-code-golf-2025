# Task 400: 対称グリッドのマスク復元

## 問題

24×24 のグリッドが与えられます。グリッド内には色付きの模様が描かれていますが、
そのうち一か所が **青 (1)** の 5×5 の正方形で塗りつぶされ、元の模様が隠されています。

グリッド全体は上下左右で対称であり、マスク以外の位置ではこの対称性が保たれています。

あなたの仕事は、対称性を利用して隠された 5×5 の模様を復元し、その部分だけを出力することです。

## 入力

- `g` : 24×24 の整数グリッド。
- 1 はマスクを表す特別な色で、他の色としては 0–9 が使用される。
- グリッドには 1 で塗られた 5×5 の連続した領域がただ一つ存在する。

## 出力

- マスクされた 5×5 の領域の本来の模様を、5×5 のグリッドとして返す。

## 復元ルール

各マスクセルについて、以下の 3 つの対称位置を調べる：

1. 左右反転位置 `(i, W-1-j)`
2. 上下反転位置 `(H-1-i, j)`
3. 上下左右反転位置 `(H-1-i, W-1-j)`

これらの位置のうち、マスクされていないセルの色がそのセルの元の色である。
すべてのセルは少なくとも一つの非マスク位置と対応している。

## 例

入力グリッドの一部（抜粋）:

```
...
```

出力（復元された模様）:

```
0 3 3 3 3
0 3 3 3 3
3 0 0 3 3
3 3 3 0 3
3 3 3 0 3
```

この出力は、対称位置から色をコピーしてマスクを復元した結果です。

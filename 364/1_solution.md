# Task 364: 連結成分の形を判定して色塗り

## 問題

0 から 9 までの整数からなるグリッド `g` が与えられる。値 3 のマスは 1 マス幅の線分で構成された図形を表し、各連結成分は必ず以下のいずれかの形状になる。

- L 字型
- コの字型（長方形の 3 辺）
- T 字型

各連結成分の形状に応じて次の色に塗り替えよ。

- L 字型: 青 (1)
- コの字型: 紫 (6)
- T 字型: 赤 (2)

向きは自由であり、その他のセルは変更しない。

## 実装方法


- リスト内包表記でa=g[i-1][j],b=g[i][j-1],c=g[i][j],d=d[i][j+1],e=d[i+1][j]としてg[i][j]を(c>0)*(a|b|d|e|(((a>0)+(b>0)+(d>0)+(e>0)>2)*4+((a>0)+(b>0)+(d>0)+(e>0)==1)*((a>0)*16+(b>0)*8+(d>0)*8+(e>0)*16)))で置き換える(範囲外は0で処理)
- 上記を10回繰り返す
- 最後にg全体に以下の処理をする
- 15,23を2に置き換える
- 27を1に置き換える
- 11,19を6に置き換える

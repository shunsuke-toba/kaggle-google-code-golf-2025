# Task 122: 3x3 ブロックの移動

## 問題文

H×W のグリッドが与えられる。各セルは整数で、0 は空白、2 は赤、3 は緑を表す。

グリッドにはちょうど一つ、2 で囲まれ中央が 3 の 3×3 のブロックが存在する。またこのブロックの中心と同じ行もしくは列に、ちょうど 2 マス離れた位置に別の緑のセルがある。

このブロックをその緑のセルに向けて 2 マス右または下へ平行移動させる。移動後の 3×3 の中心はその緑のセルとなる。移動元の赤いセルは 0 で塗りつぶし、もともとの緑のセルはそのまま残す。

## 実装方法
- 以下の操作を2回行う
- 各行をみて要素の和が8より大きい行があった場合
 - 列の数が偶数だった場合盤面全体を右に2つシフトする(左に追加する2列は一番右の2列)
 - 列の数が奇数だった場合盤面全体を右に2つシフトする(左に追加する2列は一番右の3列のうち左2列)
- 盤面を転置する

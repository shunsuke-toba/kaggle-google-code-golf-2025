# Task 195: 3×3 Gray Block Fractal

## 問題

0 と 5 からなるグリッド `g` が与えられます。値 5 のセルは必ず 3×3 の完全なブロックを形成しており，すべての 5 を含む最小の矩形は 9×9 です。この矩形を 3×3 のブロックで区切ると，5 の有無による 3×3 のパターン `P` が得られます。

出力では 9×9 のグリッドを作ります。各セル `(i,j)` を次のように決めます。

- `P` の位置 `(i//3, j//3)` が 1（=5 が存在）なら，`P` の位置 `(i%3, j%3)` の値を出力に用います（5 か 0）。
- それ以外の場合は 0 を出力します。

結果として，`P` をグレーの部分に再帰的に貼り付けた 9×9 の模様が得られます。

## 入力

- `g`: 整数の二次元配列（0 または 5）

## 出力

- 9×9 の二次元配列（0 または 5）

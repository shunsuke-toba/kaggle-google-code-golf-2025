# Task 257: 領域の重ね塗り

## 問題文

9×9 のグリッドが与えられます。中央の 1 列と 1 行には 1 が並んでおり，それによってグリッドは 4 つの 4×4 の領域に分割されています。各領域には 0 と 1 以外の色が 1 種類ずつ含まれます。

これら 4 つの領域を次の順で小さなキャンバスに重ね塗りしてください。

1. 左上
2. 右上
3. 左下
4. 右下

キャンバスは 4×4 の黒 (0) で初期化されています。ある領域を塗るとき，その領域で使われている色のセルを同じ位置に写し込みます。ただし，キャンバスで既に 0 以外の値があるセルは上書きしません。

最終的なキャンバスを出力してください。

## 入力

- 9×9 の整数グリッド `g`
- 各値は 0–9 の整数

## 出力

- 4×4 の整数グリッド

## 例

入力

```
077710404
777014400
000010004
700010000
111111111
000016660
008810000
808016006
000810000
```

出力

```
6777
7778
8084
7008
```

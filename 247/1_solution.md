## 問題

10×10 のグリッドが与えられる。各マスは 0(空白)か 1〜9 の色である。同じ色で上下左右に隣接するマスは連結成分を成す。

最大サイズの連結成分を求め、そのサイズを s とする。最大サイズの連結成分が複数ある場合は全て対象とし、各連結成分の色で高さ s の列を作って左から順に並べる。列の順序は各連結成分で最も左にあるマスの列位置が小さい順とする。

出力グリッドの高さは s、幅は対象連結成分の個数である。

## 例

入力

```
0000000000
0000000888
0040000008
0040066008
0044060000
0040066000
0000000000
0330000000
3300000000
0000000000
```

出力

```
468
468
468
468
468
```

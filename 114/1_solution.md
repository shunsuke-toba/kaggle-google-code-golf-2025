# Task 114: グリッド境界拡張と色の伝播

## 問題の考察ポイントと解き方

## 考察ポイント

1. **グリッドの拡張**: 入力グリッドを上下左右に 1 マスずつ拡張（n×m → (n+2)×(m+2)）
2. **拡張部分の初期化**: 拡張された境界部分は黒（0）で初期化
3. **色の伝播**: 元のグリッドの端にある色付きセルから、隣接する拡張部分へ色を伝播
4. **四隅の処理**: 四隅は常に黒（0）のまま

## 解くための手続き

1. 出力グリッドを(h+2)×(w+2)のサイズで作成し、すべて 0 で初期化
2. 入力グリッドを出力グリッドの中央部分(1:h+1, 1:w+1)にコピー
3. 境界への色の伝播:
   - 上端の境界: 入力の最上段の各セルの色を、その真上の境界セルに伝播
   - 下端の境界: 入力の最下段の各セルの色を、その真下の境界セルに伝播
   - 左端の境界: 入力の最左列の各セルの色を、その真左の境界セルに伝播
   - 右端の境界: 入力の最右列の各セルの色を、その真右の境界セルに伝播
4. 四隅は 0 のまま保持（伝播しない）

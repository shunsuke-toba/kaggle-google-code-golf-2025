# Task 138: 枠線への色の伸長

## 問題文

0〜9 の数値からなるグリッドが与えられます。このグリッドには、非零色で塗られた縦線が 2 本、横線が 2 本存在し、これらの線は矩形の枠を形成しています。

各線はほぼ全てのマスが同じ色で塗られており（行なら幅-2、列なら高さ-2 以上）、その他のマスは 0 など任意の値です。

### 変換手順

1. 2 本の縦線と 2 本の横線を見つけ、それらで囲まれた矩形領域を切り取ります（線を含む）。
2. その領域の内部にある非零セルのうち、色が上・下・左・右いずれかの枠線と一致するものを対象とします。
3. 対象セルから対応する枠線の方向へ同じ色で塗り伸ばし、枠線に突き当たるまで塗り続けます。
4. 得られた矩形領域を出力してください。

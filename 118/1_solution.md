# Task 118: スコア最大のクロスで灰色を水色に塗り替える

## 問題

色を表す整数グリッド `g` が与えられます。セルの値は 0 が黒、2 が赤、5 が灰、8 が水色を表します。

サイズ 7 と 5 の十字型を候補として、以下を繰り返してください。

1. 十字の中心を任意のセルに置きます。十字は上下左右に同じ長さの腕を持ち、サイズ `s` のとき腕の長さは `s//2` です。グリッド外に出る部分は無視します。
2. 十字内の各セルに対しスコアを加算します。
   - 赤(2) のセルは +1
   - 黒(0) または水色(8) のセルは -10
   - すでに選ばれたセルを再び使うと -100
3. スコアが最も高い十字を選びます。同点ならサイズが小さい方を選びます。また、一度サイズ 7 を選ぶと以降はサイズ 5 を選べません。
4. 選んだ十字の灰色(5)のセルを水色(8)に塗り替え、そのセルを使用済みにします。
5. 最大スコアが 0 以下になったら終了します。

最終的なグリッドを返してください。

## 入力

- 2 次元リスト `g` (`H×W`, 各要素は 0〜9 の整数)

## 出力

- 変換後のグリッド

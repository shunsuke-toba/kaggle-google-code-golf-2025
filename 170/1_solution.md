# Task 170: マスクに従って値を抽出

## 問題文

グリッド `g` が与えられる。`g` には二つの特徴的な領域が含まれる。

1. 最も多く出現する色 `d` で構成された領域があり、同じ大きさの正方形ブロックが `n×n` 個並んでいる。このブロック配置はマスクを表す。
2. `d` と 0 以外の値から成る `n×n` の正方形領域が別の位置に存在し、そこには様々な数字が書かれている。

マスク領域の各ブロックは同じサイズ `s×s` の正方形で、`n` は 3 または 4 である。マスク領域と数値領域は離れて配置される。

マスクの形に従って数値領域から値を取り出し、その他のセルは 0 にする。

## 出力

- `n×n` のグリッドを出力する。各セルは対応する数値領域の値か 0 である。
- マスクに対応する位置では数値領域の値をそのまま残し、マスクが無い位置は 0 とする。

## 手順

1. 色 `d` を最頻出色として求める。
2. `d` 以外の非 0 色からなる最小の正方形領域を取り出し、これを数値領域とする。
3. 数値領域の大きさを `n` とする。`d` のブロックで構成された領域を `n×n` 個のセルに分割し、各セルに `d` が存在するかでマスクを作成する。
4. マスクを数値領域に適用し、`d` がある位置の値を残し、他を 0 に置き換える。

## 例

例えば、マスクが以下のように配置され、数値領域が `[[3,1,7],[2,8,9],[3,4,6]]` のとき、マスク形状が

```
1 0 1
0 1 0
1 0 1
```

であれば出力は

```
3 0 7
0 8 0
3 0 6
```

となる。

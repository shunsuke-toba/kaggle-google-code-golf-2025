# Task 370: 斜め複製パターン

## 問題

背景色で満たされたグリッドに、色 0 のブロックが 8 近傍で連結した図形が 1 つあり、別の色で塗られたブロックが 1 つだけ存在します。この方向ブロックの位置から図形をどの斜め方向へ複製するかを決定し、図形をその色で繰り返しコピーしてください。

## ルール

1. 色 0 のブロック集合を図形とする。集合の重心を図形の中心とする。
2. 図形の中心と方向ブロックの位置関係から、進むべき斜め方向（\(\text{NE},\text{NW},\text{SE},\text{SW}\)）を決定する。
3. 図形をその方向へ平行移動する。図形同士が重ならない最小の距離だけ移動させ、以降同じ距離で繰り返す。
4. 移動先の図形は方向ブロックの色で塗る。グリッド外に出るセルは無視し、移動によってグリッド内にセルが 1 つも残らなくなった時点で終了する。
5. 既存のセルは上書きしない。

## 出力

上記の操作を行った後のグリッドを返してください。
